2. Como funciona o Machine Learning?

O Machine Learning (Aprendizado de Máquina) permite que o sistema aprenda com o passado sem ser explicitamente programado para cada situação.

Funcionamento do Script (Lógica de Front-end)

O arquivo script.js gerencia a interatividade da página sem a necessidade de recarregamento. Ele atua em três frentes principais:

1. Interceptação do Formulário

O script impede o envio padrão do formulário (e.preventDefault()), permitindo que o processamento ocorra localmente com uma animação de carregamento no botão.

2. Simulação e Gauge

Ele gera um valor probabilístico e calcula a rotação do medidor (velocímetro) usando a unidade turn do CSS para transformar porcentagem em movimento visual.

3. Gatilhos de Animação

Ao carregar a página, o script detecta elementos com a classe .reveal e adiciona a classe active, disparando as transições suaves de entrada.
